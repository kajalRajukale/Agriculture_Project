# How to start the Frontend

## First, open the correct project folder with VSCode

C:\Coding4Beginner\Workspace.Django\Agriculture_Project

## Preview the "index.html" of the Frontend

- Select folder "frontend"
- Select file  "index.html"
- Right Click and select ""Show Preview"
- A tab is opened with the frontend
- Copy the url
  http://127.0.0.1:3000/frontend/index.html
- Open a Browser Window
- Past url
  http://127.0.0.1:3000/frontend/index.html
- To view Farmers, select Top Navigation Item "Farmers"
